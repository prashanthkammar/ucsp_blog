Concept of Probability in Communication

A workshop on 'Concept of Probability in Communication', covering major concepts of the domain, was held on 3rd of January 2021, Sunday from 10AM to 12PM. The event started from the basics of probability. Correlating to topics taught on probability, their usage and application in the field of wireless and wired communication was discussed. This was followed by a Q/A session where the doubts were cleared, there by marking the end of the workshop.
Resource person: Dr. Sanjeev G, Professor, Department of ECE, PES University.

Cryptography in Projects

'Cryptography in Projects' was a two session event covering both theory as well as its applications in the field. The first session included introduction to cryptography, vulnerabilities and attacks on wired and wireless networks, TCP/IP model, security goals, services and mechanisms, Euclidian algorithm, process of GCP, ciphers and several related topics. In the second session, two projects were displayed. One of them was using a project on RSA algorithm using python programming language. Other was encryption and decryption using Hill Cipher algorithm. Overall, this event gave a head start to students who are interested in the field by covering the prerequisites.
Resource person: Anand Srivatsa, Head of SDSC and Assistant Professor, Department of ECE, NIE Mysuru.

Image Processing

"Image processing and its applications" was an event organised to give an overview of the entire subject. Starting from the basics of the subject, major areas of research in Image processing, such as Biomedical image processing, ML in Image processing, steganography and many more fields were covered. In the end, insights on projects for UG students were also given.
Resource person: Dr. Nagaraju C, Assistant Professor, Department of ECE, NIE Mysuru.

Lockdown Saga

This was an elaborate technofun event which consisted of four rounds. The first one was a brain teaser in which you had to make wise choices to reach the end of the story. The second round had image based puzzles. The next one was a virtual treasure hunt and the last round was a quiz including general aptitude related questions. This fun and learn event was entirely designed and built by the core team of UCSP.

Computer Networking Webinar

Computer Networking being one such subject which is very important as well as in current trends, a session on this was organised. The topics covered ranged from basics of the subject to some core concepts as well as research related topics.
Resource person: Dr. Gururaj H L, Associate Professor, Department of CSE, VVCE Mysuru.

Codezilla

This was a technical event which included a three hour rigorous coding competition in which the participants had to code solutions to the given problems. The language for coding was left to their choice.

Git Wit

Considering how useful Git and Git hub are to an engineering student, a workshop was organised to introduce the participants to the fundamentals of Git and Git hub, how to create an account, upload a code, make changes in an already uploaded code and more. This was conducted in collaboration with the Microsoft Student Partner Ashwin G G.
Resource Person: Shivaprasad S, 4th year ECE.

Technocazar

A technofun event in the form of treasure hunt where each team had to answer several questions on general aptitude, digital logic and concepts studied till first year of engineering in order to reach the treasure. It was conducted mainly to bridge the gap between academics and practical applications.

Machine Learning Workship

A two day workshop was conducted on Machine Learning which included basics of python, some libraries on python useful for developing machine learning projects, basics of machine learning and how to work on tensorflow. With the basics taught on the first day, a project on Digit Recognition was built on the second day.
Resource Person: Yashwanth R and Adarsh Srivatsa, 4th year ECE.

Technical Paper Writing Session

The main focus of our group is to introduce research to UG students. One way of doing this is through technical paper writing and hence, this session. It helped students understand what a technical paper is, types of papers, paper templates, methods of writing as well as its importance. It also gave them an idea of how to present a technical paper on various platforms.
Resource Person: Dr. Roopa K, Associate Professor, Dept of ECE, NIE.
